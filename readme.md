# TourneyPro - Football Tournament Manager

A Progressive Web App (PWA) that works like a native Android app — no App Store needed.

## 📦 Files
- `index.html` — Main app
- `manifest.json` — PWA manifest (makes it installable)
- `sw.js` — Service Worker (enables offline use)

## 🚀 How to Share With Others (Free Hosting Options)

### Option 1: GitHub Pages (Recommended — Free)
1. Create a free account at github.com
2. Create a new repository (e.g. "tourneypro")
3. Upload all 3 files
4. Go to Settings → Pages → Deploy from Branch → main
5. Your app is live at: `https://yourusername.github.io/tourneypro`
6. Share this link via WhatsApp, SMS, etc.

### Option 2: Netlify (Drag & Drop — Free)
1. Go to netlify.com → Sign up free
2. Drag the folder onto their deploy area
3. Get an instant URL like `https://tourneypro.netlify.app`

### Option 3: Vercel (Free)
1. Go to vercel.com → Sign up free
2. Import or drag & drop files
3. Get a shareable URL

## 📱 How Android Users Install It
1. Open the shared link in Chrome
2. Tap the 3-dot menu (⋮) in Chrome
3. Tap "Add to Home Screen"
4. Tap "Add"
5. The app icon appears on their home screen — works like a native app!

## ✨ Features
- **Multiple Tournaments** — Create as many as needed
- **3 Formats** — League (Round Robin), Groups + Knockout, Knockout Only
- **Auto Fixture Generation** — Balanced round-robin scheduling
- **Manual Fixtures** — Add custom matches anytime
- **Live Points Table** — Auto-calculates after every result
- **Form Guide** — Last 5 match results per team
- **Knockout Bracket** — Manage QF, SF, Final with penalty shootout support
- **Custom Points** — Configure win/draw/loss points freely
- **Team Colors** — Visual color coding for all teams
- **Offline Support** — Works without internet after first load
- **All data saved locally** — No account or server needed

## 🎮 How to Use
1. Tap **+ New** to create a tournament
2. Go to **Teams** tab → add all teams
3. Tap **⚡ Generate Fixtures** for auto-schedule
4. Go to **Fixtures** tab → enter results as matches are played
5. Check **Table** for live standings
6. For knockout formats, use the **Knockout** tab

## 📊 Points Configuration
Default: Win = 3pts, Draw = 1pt, Loss = 0pts
Customize when creating the tournament.
